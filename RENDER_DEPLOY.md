# Deploy The Telkom Manual Checker To Render

This folder is now Render-ready.

## What To Deploy

Use these files:

- `render_app.py`
- `telkom_batch_check.py`
- `requirements.txt`
- `render.yaml`

## Render Settings

Create a new Render Web Service from a GitHub repo containing those files.

- Runtime: Python
- Build command: `pip install -r requirements.txt`
- Start command: `gunicorn render_app:app`

The included `render.yaml` already defines those settings.

## Important Notes

- The app uploads Excel files in the browser, so it no longer depends on local Windows paths.
- The app stores uploaded files, cache, and exported reports under `DATA_DIR`.
- The included config uses `/tmp/telkom-render-checker`, which is simple but temporary.
- For a long checking session, add a Render persistent disk and set `DATA_DIR` to that disk path.
- `APP_PASSWORD` is generated by Render. Open the app with `?key=YOUR_APP_PASSWORD`.

## Workflow

1. Open the Render URL.
2. Add `?key=YOUR_APP_PASSWORD` to the URL if password protection is enabled.
3. Upload the Excel files.
4. Click `Check This Number`.
5. Type captcha when asked.
6. Export and download the reports.
